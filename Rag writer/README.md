# 🏛️ RAG Writer - 유럽 사내변호사 ACP 연구 보고서 생성기

AI 기반 자동 보고서 생성 시스템으로, 유럽 대륙법계 주요국의 사내변호사 비밀유지특권(ACP) 제도를 분석하여 한국 도입 방안을 제시하는 전문 보고서를 생성합니다.

## 📋 프로젝트 개요

본 시스템은 RAG(Retrieval-Augmented Generation) 기법을 활용하여 복잡한 법률 연구 보고서를 체계적으로 생성합니다. Google의 Gemini AI 모델을 사용하여 다단계 보고서 작성 프로세스를 자동화하며, 전문적인 법률 문서 품질을 보장합니다.

### 🎯 연구 주제
- **사내변호사 비밀유지특권(Attorney-Client Privilege, ACP)**
- **유럽 대륙법계 국가별 제도 비교 분석**
- **한국 도입을 위한 정책 제언**

## ⚡ 주요 기능

### 🔄 6단계 AI 파이프라인
1. **📂 소스 파일 읽기** - 원본 연구 자료 통합 분석
2. **📚 참고문헌 추출** - AI 기반 각주 및 참고문헌 자동 파싱
3. **📋 개요 생성** - 체계적인 보고서 구조 설계
4. **📝 초안 생성** - 상세한 보고서 본문 작성
5. **✏️ 편집장 검토** - 내용 개선 및 품질 향상
6. **✅ 최종 품질 검사** - 사실 확인 및 일관성 검증

### 🛠️ 핵심 기술
- **RAG (Retrieval-Augmented Generation)** - 외부 지식 통합
- **멀티 모델 파이프라인** - 단계별 최적화된 AI 모델 사용
- **자동 참고문헌 관리** - 79개 법률 문헌 자동 추출/정리
- **품질 보증 시스템** - 다단계 검토 및 검증 프로세스

│   ├── 📄 유럽 사내변호사 ACP 심층 분석_.md

│   └── 📄 유럽 대륙법계 주요국의 사내변호사 비밀유지권(ACP) 제도 도입 사례 및 한국 입법 방안.pdf

└── 📄 final_report.md             # 최종 확정 보고서

## 🚀 사용 방법

### 전체 파이프라인 실행 (권장)
```bash
python generate_report.py
```

### 단계별 실행 옵션
```bash
# 초안만 생성
python generate_report.py --step draft

# 기존 초안을 향상
python generate_report.py --step enhance --draft-path "draft_report_20250707_123456.md"
```

## 📊 생성 프로세스 상세

### 1단계: 소스 파일 통합
- 7개 원본 파일 자동 읽기
- 중복 내용 분석 및 통합
- 텍스트 전처리 및 정규화

### 2단계: 참고문헌 자동 추출
- AI 기반 각주 파싱 (평균 3-5개)
- 참고문헌 자동 추출 (평균 79개)
- 중복 제거 및 표준화

### 3단계: 구조적 개요 생성
- 논리적 흐름 설계
- 장/절 구조 최적화
- 한국 도입 방안 포함

### 4단계: 전문 보고서 작성
- 최소 15,000자 이상 생성
- 법률 용어 정확성 보장
- 국가별 사례 상세 분석

### 5단계: 편집장급 검토
- 내용 일관성 검증
- 중복 제거 및 통합
- 논리적 흐름 개선

### 6단계: 최종 품질 보증
- 사실 관계 확인
- 참고문헌 연결성 검증
- 용어 일관성 검사

## 📄 출력 파일 설명

### enhanced_report_YYYYMMDD_HHMMSS.md
**최종 보고서** - 완성된 연구 보고서
- 서론: ACP 제도 필요성

- 한국 도입 방안: 하이브리드 모델 제시

**생성 메타데이터** - 품질 보증 정보
- 사용 모델 및 설정
- 단계별 처리 결과
- 품질 검사 보고서
- 개선 제안사항
