학습 목표

1.1 AI 기본 개념과 가족 관계도 (15분)

세 용어는 종종 혼용되지만, 그 관계를 이해하는 것은 AI를 제대로 파악하는 데 매우 중요합니다. 간단하고 기억하기 쉬운 마트료시카 인형 비유를 사용하겠습니다.

🪆 가장 큰 인형: 인공지능(AI)

튜링 시대부터 시작된 가장 큰 꿈, 즉 인간의 지능을 어떤 방식으로든 흉내 내는 기계를 만들려는 모든 개념과 시도를 포함합니다.

  • 딥블루와 같은 '구식' 규칙 기반 시스템도 포함
  • 체스, 바둑부터 대화, 이미지 인식까지 모든 지능적 행동

🪆 중간 인형: 머신러닝(ML)

AI를 구현하기 위한 '접근법' 중 하나입니다. 모든 상황에 대한 규칙을 인간이 일일이 프로그래밍하는 대신, 기계가 데이터로부터 '스스로 학습'하게 만드는 방식입니다.

  • 지도 학습: '정답지'를 주고 배우는 방식
  • 비지도 학습: 정답 없이 데이터 속에서 패턴을 찾는 방식

🪆 가장 작은 인형: 딥러닝(DL)

머신러닝의 여러 '기술' 중 하나로, 현재 가장 강력한 성능을 보여주는 기술입니다. 인간의 뇌 구조에서 영감을 받은 '인공 신경망(Artificial Neural Networks)'을 여러 겹으로 깊게 쌓아 올린 구조를 사용합니다.

딥러닝의 성과:
  • 알파고의 승리를 이끌었음
  • 이미지 인식, 음성 인식의 혁신
  • 오늘날의 거대 언어 모델(LLM)을 가능하게 한 핵심 기술
핵심: 어떤 것이 절대적으로 '더 좋다'의 문제가 아니라, 주어진 상황에 어떤 것이 '더 적합한가'의 문제입니다. 각각은 서로 다른 문제를 해결하기 위한 도구들입니다.

1.2 생성형 AI 개요와 분류 (10분)

이전의 AI가 주로 '분석'과 '분류'에 집중했다면, 생성형 AI는 완전히 새로운 패러다임을 제시합니다.

🔍 전통적 AI (분석형)

  • 이미지 분류: "이것은 고양이입니다"
  • 스팸 탐지: "이것은 스팸 메일입니다"
  • 음성 인식: "안녕하세요"
  • 추천 시스템: "이 영화를 좋아할 것입니다"

🎨 생성형 AI (창조형)

  • 텍스트 생성: "새로운 소설을 써드립니다"
  • 이미지 생성: "상상 속 그림을 그려드립니다"
  • 음악 생성: "새로운 멜로디를 작곡합니다"
  • 코드 생성: "프로그램을 작성해드립니다"

1.4 이미지 생성 AI와 기반 기술들 (15분)

이미지 생성 AI는 현재 가장 주목받는 생성형 AI 분야 중 하나입니다. 텍스트 설명만으로 놀라운 품질의 이미지를 생성할 수 있습니다.

🎨 이미지 생성 AI의 마법

입력: "석양이 지는 바다 위에서 고래가 점프하는 모습, 판타지 스타일"

출력: 몇 초 만에 전문 아티스트 수준의 이미지 생성!

이것이 어떻게 가능할까요? 그 비밀을 파헤쳐보겠습니다.

🔧 핵심 기반 기술들

🌊 Diffusion Models (확산 모델)

작동 원리: 노이즈에서 시작해서 점진적으로 이미지를 "복원"하는 방식

단계별 과정:
  1. 완전한 노이즈 이미지에서 시작
  2. 텍스트 프롬프트를 분석
  3. 노이즈를 조금씩 제거하며 이미지 형성
  4. 수십 번의 반복을 통해 최종 이미지 완성

대표 모델: Stable Diffusion, DALL-E 2/3

장점: 높은 품질, 안정적인 학습

⚔️ GAN (생성적 적대 신경망)

작동 원리: 생성자와 판별자가 서로 경쟁하며 발전하는 구조

GAN의 구조:
  • 생성자(Generator): 가짜 이미지를 만드는 AI
  • 판별자(Discriminator): 진짜와 가짜를 구분하는 AI
  • 경쟁 학습: 서로를 속이고 찾아내며 실력 향상

비유: 위조지폐범(생성자) vs 경찰(판별자)의 끝없는 경쟁

장점: 매우 사실적인 이미지 생성 가능

🧠 VAE (변분 오토인코더)

작동 원리: 이미지를 압축했다가 다시 복원하는 과정에서 새로운 이미지 생성

VAE 과정:
  1. 인코딩: 이미지를 작은 코드로 압축
  2. 잠재 공간: 압축된 정보들이 모이는 공간
  3. 디코딩: 코드를 다시 이미지로 복원
  4. 변형: 코드를 조작해서 새로운 이미지 생성

장점: 이미지 편집과 스타일 변환에 유용

🔤 CLIP (텍스트-이미지 연결)

작동 원리: 텍스트와 이미지를 같은 공간에서 이해할 수 있게 해주는 기술

CLIP의 역할:
  • 텍스트 프롬프트를 이해
  • 이미지의 내용을 파악
  • 둘 사이의 연관성을 학습
  • 텍스트 설명에 맞는 이미지 생성 가이드

중요성: 텍스트로 이미지를 생성하는 모든 AI의 핵심 기술

1.5 LLM vs 이미지 생성 AI 비교 및 정리 (15분)

앞선 내용들을 종합하여 현재 AI 기술의 최전선을 소개합니다. 딥러닝 기술의 발전이 '생성형 AI'라는 새로운 시대를 열었습니다.

🎨 생성형 AI(Generative AI)

이전의 AI가 주로 주어진 데이터를 '분석'하고 '분류'하는 역할을 했다면:

  • 이미지가 고양이인지 개인지 판별
  • 이메일이 스팸인지 아닌지 분류
  • 음성을 텍스트로 변환

생성형 AI는 기존에 없던 새로운 콘텐츠를 '창조'하는 AI입니다:

  • 텍스트 생성 (글쓰기, 번역, 요약)
  • 이미지 생성 (그림, 사진, 디자인)
  • 음악 생성
  • 코드 생성

🧠 거대 언어 모델(LLM)

챗GPT(ChatGPT)와 같이 현재 가장 유명한 생성형 AI 도구들의 핵심 엔진이 바로 '거대 언어 모델(Large Language Model, LLM)'입니다.

LLM의 특징:
  • 거대한 규모: 수십억~수조 개의 매개변수
  • 언어 이해: 문맥과 의미를 파악
  • 다양한 작업: 번역, 요약, 창작, 코딩 등
  • 대화형 인터페이스: 자연어로 소통

📊 전통적 AI vs 생성형 AI 비교

구분 전통적 AI 생성형 AI
주요 기능 분석, 분류, 예측 생성, 창작, 변환
입력 구조화된 데이터 자연어 프롬프트
출력 카테고리, 점수, 라벨 텍스트, 이미지, 코드
사용자 경험 간접적, 백그라운드 직접적, 대화형
대표 사례 스팸 필터, 추천 시스템 ChatGPT, DALL-E

LLM과 이미지 생성 AI는 서로 다른 강점을 가지고 있으며, 각각의 기반 기술도 다릅니다. 하지만 둘 다 생성형 AI의 핵심 분야로서 빠르게 발전하고 있습니다.

🔄 LLM vs 이미지 생성 AI 심화 비교

구분 LLM (텍스트 생성) 이미지 생성 AI
주요 기반 기술 Transformer, Attention Diffusion, GAN, VAE
학습 데이터 텍스트 (책, 웹페이지, 대화) 이미지-텍스트 쌍
출력 형태 순차적 텍스트 생성 전체 이미지 동시 생성
제어 방식 프롬프트, 대화형 텍스트 프롬프트, 스타일 파라미터
계산 요구사항 높음 (대화 유지) 매우 높음 (이미지 생성)
활용 분야 글쓰기, 번역, 코딩, 분석 디자인, 아트, 마케팅, 게임
미래 전망: LLM과 이미지 생성 AI는 점점 더 통합되고 있습니다. GPT-4V처럼 텍스트와 이미지를 모두 이해하는 멀티모달 AI, 그리고 DALL-E 3처럼 더 정교한 텍스트 이해력을 가진 이미지 생성 AI가 등장하고 있습니다. 앞으로는 텍스트, 이미지, 음성, 비디오를 모두 처리할 수 있는 통합 AI 시대가 올 것으로 예상됩니다.
