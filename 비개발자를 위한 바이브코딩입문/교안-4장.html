비전공자를 위한 AI 기초와 바이브 코딩 (4시간 커리큘럼)

4장. 실습: VS Code + Google Gemini Code Assist (1시간)

목표

  • 에디터에서 “대화→코드→검증” 루프를 체험하고, 업무형 미니 과제를 완주

사전 준비(5분)

  • Visual Studio Code 설치
  • Google 계정 준비(로그인 가능 상태)

설치/로그인(10분)

  • VS Code 실행 → 확장(Extensions) → ‘Gemini Code Assist’ 검색·설치
  • 상태바 로그인 안내에 따라 Google 로그인 완료

핵심 기능 체험(20분)

  • 주석으로 요구사항 작성 → 코드 제안(자동완성/스니펫/치트시트 생성)
  • 코드 설명/리팩터링/테스트 코드 생성 요청 → 변경점 적용·리뷰
  • 오류 메시지 붙여넣기 → 원인 가설·수정 제안 받기 → 재실행

미니 프로젝트(25분)

  • A안(문서팀 친화): “요약 도우미” CLI/노트북 만들기(입력 문단 → 핵심 3줄 + 키워드)
  • B안(데이터팀 친화): “CSV 리포트” 스크립트(평균·최댓값·간단 그래프 코드 제안)
  • 심화 옵션: 결과를 표/JSON으로 내보내기, 프롬프트 템플릿화

평가 루브릭·체크리스트(5분)

  • 문제 분해가 명확했는가? 출력 형식을 요구했는가? 결과 검증 루프가 있었는가?
  • 난해할 때 반대로 질문하도록 유도했는가? 재현 가능한 산출물(스크립트/노트) 남겼는가?
